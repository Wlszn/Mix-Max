# Mix & Max — Event Ticketing Platform

##  Project Overview
Mix & Max is a full-stack eCommerce web application that allows users to discover, reserve, and purchase tickets for concerts and live events. The platform provides an intuitive interface for browsing events, selecting seats in real time, and completing secure bookings.

The application includes both a **user-facing interface** for customers and an **administrative dashboard** for managing events, venues, and ticket inventory.

##  Features

###  Event Search & Filtering
Search events by artist, venue, city, date, and genre with dynamic AJAX updates.

###  Real-Time Seat Selection
Select seats based on availability with protection against double-booking.

###  Session-Based Cart
Add tickets to a cart that persists across sessions and logins.

###  Booking & E-Ticket Generation
Secure checkout with booking confirmation and downloadable PDF tickets.

###  Admin Dashboard
Admins can manage events, venues, and tickets using CRUD operations with role-based access control and 2FA.

###  Internationalization
Supports multiple languages (e.g., English and French).

###  Flash Messaging
Displays real-time feedback for user actions (success, error, etc.).
